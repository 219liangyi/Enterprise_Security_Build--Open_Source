# 安装
## 解决依赖
```
yum install flex bison phantomjs -y
```

## 安装
```
git clone https://github.com/WhaleShark-Team/cobra.git && cd cobra
pip install -r requirements.txt
python cobra.py --help
```
# 使用
## 规则文件夹
```
cobra/rules  # 之前cobra并没有开源规则，但是现在的版本中已经有部分检测规则了
```
