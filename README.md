# Enterprise_Security_Build--Open_Source
## 甲方企业安全建设开源之路

本项目主要是编写一些在甲方企业安全建设中使用开源&破解软件的文章，也会提出一些安全建设的建议与踩坑的地方。

* 欢迎大家一起入坑，本项目将是一个长期的项目，如果有志同道合朋友一起加入，可微信我：Lzero2012，或者直接在Issues上留言讨论。
* 不断踩坑不断强大自我！！！
